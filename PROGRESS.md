# 项目进度与规划 (Project Progress)

本文档用于记录 `Terminal Fishing Game` 的开发进度、已完成的里程碑以及未来的迭代规划。

## 🟢 当前状态 (Current Status)
**版本**: v1.0.0 (基础可玩版本)

项目已经完成了核心游戏循环的搭建，具备完整的图形渲染、动画系统及 QTE 钓鱼交互机制。游戏运行稳定，无明显内存泄漏或渲染闪烁问题。

## 🚀 未来规划 (Future Roadmap)

### v1.1.0 规划：体验打磨 (Polish & Polish)
- [ ] **音效系统**：如果操作系统支持，接入简单的 `curses.beep()` 或基于 `os` 的终端提示音系统，增强咬钩时的反馈。
- [ ] **视觉增强**：增加鱼钓上水面时的挣扎动画，或者飞回玩家手中的轨迹动画。
- [ ] **动态难度**：随着分数的增加，稀有鱼的出现概率提升，但 QTE 反应时间缩短。

### v1.2.0 规划：成长与存档系统 (Progression)
- [ ] **持久化存档**：使用 `json` 格式记录玩家的最高分、钓鱼数量统计，并在启动时读取。
- [ ] **图鉴系统**：增加一个菜单界面，查看所有已解锁（钓到过）的鱼类详情。
- [ ] **商店与装备**：增加金币系统，玩家可使用金币购买更好的鱼竿（延长抛竿距离）、鱼饵（提升稀有鱼咬钩率）。

### 性能与底层优化 (Tech Debt)
- [ ] 将渲染引擎进一步抽象，隔离 `curses`，以便未来可能支持 `blessed` 或跨平台兼容性更好的库。
- [ ] 添加完善的单元测试（尤其是概率引擎和物理计算模块）。
